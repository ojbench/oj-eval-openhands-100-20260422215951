## Assignment Requirements

### Problem Description

整数文明的高塔被因式分解武器粉碎之后，uint64_t们能否使用残骸重建自己的家园？

给定$m$个正整数，依次判断每一个正整数所有真因数（不包含其本身的因数）之和是否等于其自身。

### Input Format

第一行输入一个正整数 $m$，为判断数字的个数。

第二行输入$m$个正整数，以空格间隔。

### Output Format

共输出$m$行，第$i$行代表对第$i$个整数的判断。

若相等则输出 1，不相等则输出 0.

### Samples

#### Sample 1

**Input:**
```
5
4 14 4 5 6
```

**Output:**
```
0
0
0
0
1
```

### Data Constraints

$1\leq m\leq 10$，$1\leq n\leq 10^{12}$
